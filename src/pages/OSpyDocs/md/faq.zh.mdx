### O-Spy 和 PageSpy 是什么关系？#difference

O-Spy 是基于 [PageSpy](/) 及其插件生态的能力构建而来，具体而言，它打包了以下三个依赖：

- `@huolala-tech/page-spy-browser`；
- `@huolala-tech/page-spy-plugin-rrweb`；
- `@huolala-tech/page-spy-plugin-data-harbor`；

两者的区别主要体现在：

|              | O-Spy        | PageSpy |
| ------------ | ------------ | ------- |
| 是否需要部署 | 否           | 是      |
| 在线实时调试 | ❌            | ✅       |
| 离线回放调试 | ✅            | ✅       |
| 上传数据     | 🟡 需自行处理 | ✅       |
| 下载数据     | ✅            | ✅       |

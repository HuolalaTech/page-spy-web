## Install#install

:::code-group

```bash yarn
yarn global add @huolala-tech/page-spy-api@latest && page-spy-api
```

```bash npm
npm install -g @huolala-tech/page-spy-api@latest && page-spy-api
```

:::

After completion, open the browser and visit http://localhost:6752 to access the service.

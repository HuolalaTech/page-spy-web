### 安装#install

```bash
docker run -d --restart=always -v ./log:/app/log -v ./data:/app/data -p 6752:6752 --name="pageSpy" ghcr.io/huolalatech/page-spy-web:latest
```

执行完成后，打开浏览器访问 http://localhost:6752 即可访问服务。

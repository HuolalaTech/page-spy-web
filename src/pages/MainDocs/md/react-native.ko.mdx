## 시작하기#start-using

### 1단계#step-1

프로젝트에 의존성을 설치합니다:

:::code-group

```bash yarn
yarn add @huolala-tech/page-spy-react-native@latest
```

```bash npm
npm install @huolala-tech/page-spy-react-native@latest
```

:::

### 2단계#step-2

진입점 파일에서 SDK를 가져오고 인스턴스화합니다. [설정 옵션]({VITE_SDK_RN_REPO})을 통해 SDK 동작을 사용자 정의할 수 있습니다:

```js
import PageSpy from '@huolala-tech/page-spy-react-native';

new PageSpy({
  api: "{deployPath}",
  enableSSL: {enableSSL},
})
```

`@react-native-async-storage/async-storage`를 사용하는 경우, 독립된 [플러그인]({VITE_PLUGIN_RN_STORAGE})을 제공합니다:

```js
import PageSpy from '@huolala-tech/page-spy-react-native';
import RNAsyncStoragePlugin from '@huolala-tech/page-spy-plugin-rn-async-storage';

PageSpy.registerPlugin(new RNAsyncStoragePlugin())
new PageSpy({
  api: "{deployPath}",
  enableSSL: {enableSSL},
})
```

이상이 React Native 프로젝트에 PageSpy를 연동하는 전체 과정입니다. 연동이 완료된 후 상단 메뉴의 <Tooltip title="메뉴는 기본적으로 숨겨져 있으며, 배포 후 표시됩니다" color="purple"><a href="javascript:void(0)">디버그 시작</a></Tooltip>을 클릭하여 사용하세요!
